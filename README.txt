Video Embed Field Tokens
=======================

What Is This?
-------------

This is a module which exposes additional Video Embed Field data as
tokens:

* Video Embed Player URL - The URL of the embedded player for Video
  Embed field.  Can be used with Metatag, such as for Twitter Media
  player URL.
* Video Embed Player Width - The width of the embedded player for Video
  Embed field.  Can be used with Metatag, such as for Twitter Media
  player width.
* Video Embed Player Height - The height of the embedded player for
  Video Embed field.  Can be used with Metatag, such as for Twitter
  Media player height.
